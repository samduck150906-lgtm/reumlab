■ 사이트 소유권 인증용 HTML 파일 (검색엔진 등록용)

【구글 서치콘솔】
1. search.google.com/search-console → 속성 추가 → URL 접두어 입력 (https://reumlab.com)
2. 소유권 확인 방법에서 "HTML 파일" 선택
3. 구글이 알려주는 파일명(예: google4a1b2c3d5e6f.html)으로 이 폴더의 파일 이름 변경
   또는 그 파일명으로 새 파일 만들고, 구글이 알려준 한 줄 내용만 넣기
4. 내용 예: google-site-verification=xxxxxxxxxxxxx/
5. 배포 후 서치콘솔에서 "확인" 클릭

【네이버 서치어드바이저】
1. searchadvisor.naver.com → 사이트 등록
2. "HTML 파일 업로드" 선택 시: 네이버가 알려준 파일명으로 파일 생성 후 내용 붙여넣기
3. 또는 "메타 태그" 방식이면 public/index.html 의 <head> 안에 네이버가 준 meta 태그 한 줄 추가
